Why ship a dedicated terminal?

Secure defaults

We isolate renderer processes, apply restrictive navigation policies, and inject a modern content security policy before remote pages load.

First-party experience

The terminal wraps the official BreakoutProp interface, so users authenticate and trade against the same endpoints they already trust in the browser.

Cross-platform delivery

Build installers with Electron Forge for Windows, macOS, or Linux without rewriting your web application.

Prefer the browser?

You can still access the BreakoutProp terminal online. Launch the hosted experience directly in your browser if you do not need the native shell.
